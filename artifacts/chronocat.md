# 時之貓苑 · Chronocat

> *每至整點，森林深處便浮現一隻新貓。*

## 典故

大魔導師 Yaze 初抵異世界那年，特別思念故鄉的橘貓。
他施下「時之召貓」咒文，希望每個整點都能看見一次貓。
至今已累積數千隻，各自攜帶一段小故事。

## 效能

GitHub Actions 每小時自動呼叫 Gemini API 生成 AI 貓圖，
發布至 Telegram 頻道 [@catime_yaze](https://t.me/catime_yaze)
並在 [Gallery](https://yazelin.github.io/catime/) 永久歸檔。
擁有 103 種藝術流派（從浮世繪到賽博龐克到刺繡微縮）。
