# 冥想第三眼 · AgentPulse

> *於螢幕頂端懸浮的第三眼，凝視所有精靈的工作狀態。*

## 典故

召喚師身邊精靈眾多，各於不同介面借身行事。
為使召喚師不致錯過任何一道氣息，
造此第三眼懸於視界之頂，
跨界接線，精靈睜眼、閉目、沉思，皆被觀照。

## 效能

Tauri v2 跨平台 Dynamic Island，同時監看四家 CLI 的 session 狀態：
- Claude Code
- Gemini CLI
- Codex CLI
- GitHub Copilot

是跨 CLI hook 實作的最佳參考實作。
