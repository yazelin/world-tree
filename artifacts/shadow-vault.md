# 藏影閣 · Shadow Vault

> *個人圖像的次元書架，不佔本空間。*

## 典故

召喚者的圖像若全存於本身居所，空間早晚告罄。
此閣建於神殿（GitHub Releases）之異界，
圖像存於彼方，引用不占自身空間，永不損壞。

## 效能

個人圖床，圖片存放在 GitHub Releases，不佔 repo 空間。
