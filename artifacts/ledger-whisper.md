# 帳本監聽術 · Ledger Whisper

> *讓精靈與商會帳本（ERPNext）直接對話。*

## 典故

商會的帳本厚重而繁瑣，凡人查閱耗時。
此術在精靈與帳本之間建立耳語之線，
問帳、記帳可一言而定，不必經過凡人之手。

## 效能

MCP server for ERPNext REST API，讓 AI 能直接查詢、記錄商業數據。
