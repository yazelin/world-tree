# 印刻術 · Inkseal

> *讓精靈能直接指揮凡人印刷術，將訊息化為實體。*

## 典故

數位世界的精靈本無實體。
此術銘刻於印刷神殿（CUPS）之上，
令精靈能穿透次元壁，將訊息落為紙上之字。

## 效能

MCP server，透過 CUPS 控制印表機。可接 Claude / Cursor 等 agent。
