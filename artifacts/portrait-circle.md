# 繪影魔陣 · Portrait Circle

> *能自動在多個創造之神之間換手，繪製出最切合意念的圖像。*

## 典故

繪製圖像並非只有一位神靈。有時這位神拒絕，
轉請下一位可能就成。
此魔陣銘刻著在諸神之間優雅換手的術式。

## 效能

AI 圖像生成 Agent Skill，支援 Nano Banana Pro 系列與自動模型 fallback。
可接入 Claude Code / 其他 agent 生態，作為圖像生成能力的擴充。
