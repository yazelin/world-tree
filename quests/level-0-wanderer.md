# Level 0 · 旅人（Wanderer）

> 這是你現在的位置。你使用 AI，但 AI 不認得你。

## 你的現狀

- 每次對話都是冷啟動
- 你從一個 session 跳到另一個
- 你重複解釋自己的 codebase、偏好、目標
- 你覺得 AI 「還可以」，但總是少了什麼

## 這一關的挑戰

沒有實作任務 — 只有**覺察任務**。

請在接下來一週，注意：
- 每次開啟 Claude / Gemini / Cursor 時，你花多少時間讓它「跟上你」？
- 哪些事你重複解釋了超過 3 次？
- 如果 AI 記得你上週的結論，事情會不同嗎？

## 通關條件

你發現：**這種關係可以不是這樣的**。
你願意嘗試一種新的方式。
