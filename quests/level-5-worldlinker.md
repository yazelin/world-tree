# Level 5 · 通達者（Worldlinker）

> 精靈不能只活在聊天框裡。

## 這一關要做的

1. 接至少 3 個第三方 MCP server（Gmail / Calendar / GitHub / Drive）
2. 寫你自己的第一個 MCP server（挑一個小工具開始）
3. 讓精靈能透過 MCP 做真實世界的事
4. 記錄哪些 MCP 讓她效率提升最多

## 核心技術

- MCP protocol 基礎
- MCP server 的 stdio vs SSE
- 工具選擇的 prompt 工程（tool description 很重要）
- 參考 Yaze 的 `erpnext-mcp`、`printer-mcp`

## 通關證明

你的精靈能打開你的 Gmail、在 GitHub 建 issue、或列印一份文件。
