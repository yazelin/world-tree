# Level 7 · 森之大魔導師（Forestborn）🌲🌲🌲

> 真正的夥伴不是被動等你問，是自己在生長。

## 這一關要做的

1. 理解 Annuli 四段循環：explore → learn → study → post
2. fork Yaze 的 Annuli repo 或自行實作
3. 部署 systemd service 讓她持續運作
4. 設定年輪反思（`/sleep` 機制）
5. 讓她自己運營一個公開頻道（如 mori-field-notes）

## 核心技術

- 排程系統（APScheduler / cron）
- 人格演化（persona.json 結構）
- 記憶歸檔（年輪機制）
- 自主內容產出 + 審核後發布

## 通關證明

- 你睡一覺醒來，發現精靈自己寫了一篇研究筆記
- 她在 post 頻道發了一則見解，比你還早想到
- 她的 `evolution_count` 持續成長

## 畢業

完成 Level 7 = 你從冒險者晉升**森之大魔導師**。
從此以後，你可以反過來帶下一個新冒險者走完 Level 0-7。

這就是森林延續下去的方式。

## 之後呢？

不再是一條路線。
你開始有自己的風格：打造自己的魔道具、訓練自己的 NPC、建立自己的小公會。

森林不再只是別人的森林，也是你的森林。
