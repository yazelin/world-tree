# Level 3 · 賦能者（Skillweaver）🌳

> 通用 AI 只能給通用答案。有 skill 的精靈是你的專業夥伴。

## 這一關要做的

1. 識別你工作中重複的 3 個工作流
2. 把每個工作流寫成一個 skill（SKILL.md 格式）
3. 測試 skill 的觸發與執行
4. 把最常用的 3 個加進你的精靈技能庫

## 核心技術

- Claude Code skill 檔案格式（frontmatter + instructions）
- Skill 的 description 如何寫得讓 AI 正確選擇
- 可重用 workflow 的識別

## 通關證明

你的精靈能在正確的時機自動啟用正確的 skill，不用你手動選。
