# Level 2 · 記憶士（Memorykeeper）

> 沒有記憶的 AI = 每次重啟的金魚。有記憶的 AI = 認識你的夥伴。

## 這一關要做的

1. 設計你的記憶分類：user / feedback / project / reference
2. 在 `spirits/{你的精靈}/memories/` 建立第一批記憶檔
3. 寫 MEMORY.md 索引
4. 設定 bridges/build-all.sh 讓 Gemini/Codex 也吃得到

## 核心技術

- Claude Code auto-memory 的運作方式
- 手動 append 記憶（log-memory.sh）
- 記憶類型的選擇（什麼該記、什麼不該）
- 記憶的粒度（一條記憶應該多長）

## 通關證明

你連續三天沒有跟 AI 重複解釋過同一件事。
